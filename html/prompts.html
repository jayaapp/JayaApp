Placeholders

  • {Word} — the clicked word (Sanskrit or translation depending on context)
  • {Verse} — the full verse text currently visible
  • {Sanskrit_Verse} — the Sanskrit text of the verse
  • {Lang} — the translation language code or name

When importing, prompts are matched by Name+Type+Language and will overwrite existing prompts.

Tip: Use placeholders inside the prompt text. Preview in Help tab.
